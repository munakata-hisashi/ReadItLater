# あとで読むアプリ

## アプリ名（仮）
URLブックマーク＋要約翻訳アプリ

## 概要
モバイルアプリからWebページのURLを保存し、  
サーバー側でページの内容を要約、翻訳して、アプリ上で要約文、翻訳文を閲覧できるサービス。

## ユーザーの主な機能
- URLを保存
- 保存はアプリからURLを入力するのと、iPhoneのブラウザアプリから共有して保存する
- 保存したURL一覧をリスト表示（ページタイトル付き）
- 保存したURLを削除できる
- 各URLの要約結果を閲覧
- 各URLの翻訳結果を閲覧

## データの管理
- URLはCore data with CloudKitで保存する
